## markdown tips:
- 最后我统一转成 `.docx` 上传 `teambition`
- 注意文件名尽量不要包含空格与特殊字符（包括md文件与图片文件）
- 引用图片在章节目录下创建`images`子文件夹，然后上传
- 关于数学公式，`latex` 是个好东西，可是 `gayhub` 不兹瓷，不过pandoc兹瓷，所以大家可以写着（最好顺带附一张公式的截图），这样`github`和最后的`doc`文件都能很优雅，~~懒的话就贴张图吧~~
